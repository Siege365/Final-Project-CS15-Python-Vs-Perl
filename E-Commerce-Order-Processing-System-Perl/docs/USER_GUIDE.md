# User Guide

## E-Commerce Order Processing System - Perl

Complete guide for using the e-commerce system.

## Getting Started

### Login

1. Navigate to http://localhost:3000
2. Enter credentials
3. Click "Login"

**Default Accounts:**

- Admin: admin / admin123
- Staff: staff / staff123
- Customer: customer / customer123

### Registration

1. Click "Register here" on login page
2. Fill in: Username, Email, Password
3. Confirm password
4. Click "Register"
5. Login with new account

## For Customers

### Browsing Products

1. Click "Products" in navigation
2. View product grid with:
   - Product name
   - Category
   - Price
   - Stock level
   - Description

### Searching Products

1. Use search box at top
2. Enter product name or keyword
3. Click "Search"

Or filter by category:

1. Select category from dropdown
2. Click "Search"

### Adding to Cart

1. Browse products
2. Set quantity (if needed)
3. Click "Add to Cart"
4. View cart counter in navigation

### Shopping Cart

1. Click "Cart" in navigation
2. Review items:
   - Product details
   - Quantity
   - Price per item
   - Item total
3. See cart summary:
   - Subtotal
   - Tax (8%)
   - Shipping ($5 or FREE over $100)
   - Grand Total

### Checkout

1. In cart, scroll to "Checkout" section
2. Select payment method:
   - Credit Card
   - Debit Card
   - PayPal
   - Cash on Delivery
   - Bank Transfer
3. Enter shipping address
4. Click "Place Order"

### Viewing Orders

1. Click "Orders" in navigation
2. See your order history
3. Click "View" to see order details

### Order Details

View complete order information:

- Order number
- Status (pending, processing, shipped, delivered)
- Customer information
- Shipping address
- Order items
- Pricing breakdown

## For Staff/Admin

### Dashboard

View key metrics:

- Total products
- Low stock alerts
- Total orders
- Recent orders table

### Product Management

**View All Products:**

1. Navigate to Products
2. See all active products
3. Use search and filters

**Monitor Stock:**

- Dashboard shows low stock items
- Product cards indicate stock levels

### Customer Management

1. Click "Customers" in navigation
2. View customer list with:
   - Name
   - Email
   - Phone
   - Location
   - Join date

### Order Management

1. Click "Orders"
2. View all orders (not just your own)
3. See customer names
4. Filter and search orders
5. View details
6. Update order status (if admin)

### Reports & Analytics

1. Click "Reports"
2. View metrics:
   - Total revenue
   - Total orders
   - Average order value
3. See orders by status
4. View top customers:
   - Customer name
   - Order count
   - Total spent

## Tips & Tricks

### Shopping

- Orders over $100 get FREE shipping
- Check stock before adding to cart
- Review cart before checkout
- Save shipping address for reference

### Navigation

- Use navigation bar for quick access
- Cart shows item count
- Username displayed in navigation
- Logout always available

### Product Search

- Search by name, description, or SKU
- Use category filter for browsing
- Check multiple categories

### Order Tracking

- Orders start as "pending"
- Progress through: processing → shipped → delivered
- Check order history regularly
- View detailed breakdowns

## Troubleshooting

### Cannot Add to Cart

- Check if product is in stock
- Ensure quantity is valid
- Try refreshing page

### Order Not Showing

- Check Orders page
- Verify you're logged in
- Ensure order was placed successfully

### Search Not Working

- Check spelling
- Try different keywords
- Use category filter instead

### Cart Empty After Adding

- Check if you're logged in
- Session may have expired
- Try adding again

## Security

### Password Tips

- Use strong passwords
- Don't share credentials
- Change password regularly

### Account Safety

- Logout when done
- Don't use on public computers
- Keep login details private
